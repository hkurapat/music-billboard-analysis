# Music & Billboard Data Analysis

Data analysis exploring Billboard Hot 100 chart patterns and global music trends using R.

## What I Analyzed
- Billboard Hot 100 chart trajectories (2000 season)
- Which songs improved, declined, or dropped off the chart by week 6
- Fastest and slowest songs to reach #1
- Global artist locations, familiarity, and "hotness" from the Million Song Dataset
- Geographic distribution of genres (rock, pop, blues, jazz, metal)

## Tools Used
- R / Quarto
- ggplot2, tidyverse
- Datasets: tidyr::billboard, Million Song Dataset
